Cuando escribí la primera edición de este libro entre 2011 y 2012, había menos recursos disponibles para aprender a realizar análisis de datos en Python. Esto se debía en parte al dilema del huevo y la gallina; muchas bibliotecas que ahora damos por sentadas, como pandas, scikit-learn y statsmodels, eran relativamente inmaduras en aquel entonces. Ahora, en 2022, existe una creciente bibliografía sobre ciencia de datos, análisis de datos y aprendizaje automático, que complementa los trabajos previos sobre computación científica de propósito general dirigidos a científicos computacionales, físicos y profesionales de otros campos de investigación. También existen excelentes libros sobre cómo aprender el lenguaje de programación Python y convertirse en un ingeniero de software eficaz.
